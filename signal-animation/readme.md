# 信号機点滅のアニメーション

- アニメーション後の状態を保持する

`animation-fill-mode`の値を`forwards`にすると、最後の状態を保持する

- アニメーションの再生回数を設定する

`animation-iteration-count`の値に、数値かinfiniteを設定する

> direction fill-mode」の順に記述し、省略された値は初期値となります。
